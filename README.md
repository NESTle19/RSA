# RSA Шифрование на Go

## Описание

Данная программа реализует базовое RSA шифрование на языке Go. Она позволяет:
- Находить секретный ключ по открытому ключу и модулю
- Шифровать числа с использованием открытого ключа
- Расшифровывать числа с использованием секретного ключа

## Принцип работы

Алгоритм RSA основан на сложности факторизации больших чисел. Программа использует следующие математические операции:

1. **Разложение на простые множители** - функция `Prime()` находит простые множители числа
2. **Расширенный алгоритм Евклида** - функция `methodGaus()` находит обратный элемент по модулю
3. **Быстрое возведение в степень по модулю** - функция `Enc()` реализует эффективное вычисление

## Структура программы

### Основные функции

| Функция | Описание | Параметры | Возвращаемое значение |
|---------|----------|-----------|---------------------|
| `Prime(x int) []int` | Разложение числа на простые множители | `x` - число для разложения | Срез простых множителей |
| `secret_key(open_key, mod int) int` | Вычисление секретного ключа | `open_key` - открытый ключ, `mod` - модуль | Секретный ключ или -1 при ошибке |
| `Enc(orig int, pow int64, mod int) int` | Возведение в степень по модулю | `orig` - основание, `pow` - степень, `mod` - модуль | Результат вычисления |
| `encrypt(text, openKey int64, mod int) int` | Шифрование текста | `text` - текст для шифрования, `openKey` - открытый ключ, `mod` - модуль | Зашифрованное число |
| `decrypt(encryptedText, secretKey int64, mod int) int` | Расшифровка текста | `encryptedText` - зашифрованный текст, `secretKey` - секретный ключ, `mod` - модуль | Расшифрованное число |

При запуске программа предлагает три варианта действий:

1. **Зашифровать сообщение** - только шифрование с использованием открытого ключа

## Математическое обоснование

### Алгоритм RSA

1. **Выбор ключей**:
   - Выбираются два простых числа `p` и `q`
   - Вычисляется модуль `n = p * q`
   - Вычисляется функция Эйлера `φ(n) = (p-1) * (q-1)`
   - Выбирается открытый ключ `e`, взаимно простой с `φ(n)`
   - Вычисляется секретный ключ `d` из условия `e * d ≡ 1 (mod φ(n))`

2. **Шифрование**:
   - `c = m^e mod n`, где `m` - исходное сообщение

3. **Расшифровка**:
   - `m = c^d mod n`, где `c` - зашифрованное сообщение

### Особенности реализации

- Используется расширенный алгоритм Евклида для нахождения обратного элемента
- Реализовано быстрое возведение в степень для эффективных вычислений
- Поддерживается работа с небольшими числами для демонстрационных целей

## Ограничения

- Программа работает только с целыми числами (не с текстом)
- Для реального использования требуется работа с большими числами (1024-2048 бит)
- Модуль должен быть произведением двух простых чисел
- Открытый ключ должен быть взаимно прост с функцией Эйлера

- `open_key=7, mod=33` → секретный ключ = 3
- `open_key=5, mod=21` → секретный ключ = 5
- `open_key=11, mod=35` → секретный ключ = 23
